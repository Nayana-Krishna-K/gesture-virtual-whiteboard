Gesture-Based Virtual Whiteboard System

A computer vision-based system that enables users to draw on a digital canvas using hand gestures captured through a webcam, eliminating the need for physical input devices.

Problem Statement

Traditional digital writing tools such as stylus-based tablets and smart boards are expensive and require additional hardware. This limits accessibility for many users, especially in online learning environments.

Proposed Solution

The proposed system provides a cost-effective and touchless solution using computer vision. It allows users to draw and interact using simple hand gestures without requiring any external devices.

System Methodology

Input

Webcam captures real-time video of hand movements.

Processing

MediaPipe detects hand landmarks and OpenCV processes gestures.

Output

Drawing, erasing, and color selection are displayed on a virtual canvas.

Key Features

Real-time gesture-based drawing
Touchless interaction
Gesture-controlled erasing
Color selection interface
Dynamic brush size control
Cost-effective solution

Results

The system successfully detects hand gestures and allows smooth real-time drawing. It performs effectively under proper lighting conditions and provides an interactive user experience.

Applications

Online teaching, virtual meetings, content creation, and digital drawing.

Future Scope

Future improvements include handwriting recognition for clean text conversion, more color and pen options, improved zoom functionality, smoother drawing, and enhanced user interface.
