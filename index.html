Gesture-Based Virtual Whiteboard System

A computer vision-based system that enables users to draw on a digital canvas using hand gestures captured through a webcam, eliminating the need for physical input devices.

Problem Statement

Traditional digital writing tools such as stylus-based tablets and smart boards require additional hardware and increase cost. Existing gesture-based systems often suffer from accuracy issues and complex setup, limiting accessibility.

Existing Solutions

Stylus-based tablets require physical devices, while gesture systems often face latency and accuracy challenges.

Research Gap

A simple, low-cost and real-time gesture-based system using only a webcam can improve accessibility and usability.

System Methodology

Input

Webcam captures real-time hand movement

Processing

MediaPipe detects landmarks, OpenCV tracks gestures

Output

Drawing displayed on virtual canvas

Results

The system successfully detects hand gestures and enables smooth real-time drawing with good accuracy under normal lighting conditions.
