Gesture-Based Virtual Whiteboard System

A computer vision-based system that enables users to draw on a digital canvas using hand gestures captured through a webcam. The system eliminates the need for physical input devices and provides a cost-effective, touchless solution for digital interaction.

Problem Statement

Traditional digital writing tools such as stylus-based tablets and smart boards are expensive and require additional hardware. This reduces accessibility, especially in online learning and virtual collaboration environments.

Proposed Solution

The proposed system provides a cost-effective and touchless solution using computer vision. It enables users to draw and interact using simple hand gestures without requiring any external devices.

System Methodology

Input

Live webcam video is used to capture real-time hand movements.

Processing

MediaPipe detects 21 hand landmarks and OpenCV processes gestures based on finger positions.

Output

Drawing, erasing, and color selection actions are displayed on a virtual canvas in real time.

Key Features

Real-time gesture-based drawing
Touchless interaction
Gesture-controlled erasing
Color selection interface
Dynamic brush size control
Cost-effective solution

Results

The system successfully detects hand gestures and enables smooth real-time drawing on a virtual canvas. It performs effectively under proper lighting conditions and provides an interactive user experience.

Applications

Online teaching, virtual classrooms, content creation, virtual meetings, and digital drawing applications.

Future Scope

Future improvements include handwriting recognition to convert rough writing into clean text, more color and pen options, improved zoom functionality, smoother drawing, and enhanced user interface.
